Linux Help Bot

23 August 2021

The Linux Help Bot (@linuxhelpbot) is a Twitter bot written in Python using the tweepy module created to support people who are learning Linux.

The bot retweets tweets that contain the hashtags #LinuxHelp or #LinuxQuestion. The idea of the bot is that Twitter users with Linux related questions can use the hashtags for their tweet to get more visibility, and Twitter users who are interested in answering Linux related quesstions can follow teh bot to have tweets containing Linux questions retweeted to their feed.

If you would like to prevent the bot from retweeting your tweets, feel free to block @linuxhelpbot.

I plan to add more functionality to the bot in the future so that it will like the tweets it retweets and follow the users that follow it.
