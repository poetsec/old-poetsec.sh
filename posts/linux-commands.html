Useful Linux Commands

12 August 2021

This is a running collection of Linux commands I've found are essential and use frequently.

ls Lists files and folders in your current directory. The -a option displays all files, including hidden ones. ls can be used with a file path to list the contents of that specific directory. -l is the long format option and will display details along with each entry.

cp Copy command. It's used for copying files from one location to another. Multiple files can be copied by adding multiple file names. For example: cp file1 file2 file3 target_directory. If the target directory has existing files with the same name as the ones you are trying to copy, the original files will be overwritten.

pwd Shows you the directory path you are currently in.

cd Command for navigating directories in the terminal. Syntax is cd directory_name to go down, and cd .. to go up one level.

mv Similar to cp but used for moving instead of copying. Essentially a cut-paste command. The syntax for this command is: mv file target_directory.

mkdir This command is used to create a new directory. Syntax is mkdir [option] directory_name_or_path.
